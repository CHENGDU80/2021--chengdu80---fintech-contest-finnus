# README

## Folders

There are 3 folders

./Feature Engineering: contains codes for data pre-processing and feature engineering for each raw data file
./Model: machine learning model used to generate the risk class result for testing data provided
./Prediction Result: predicted risk class using the model (.csv file)


# Early warning system designed for Risk Management

## Overview  
FinNUS is a platform that aims to provide early warning signals of potential corporate vulnerabilities. It leverages on big data and machine learning technology to provide user interpretable risk analysis which enable them to make informed business decision.

An interpretable way of predicting the potential corporate vulnerabilities.
